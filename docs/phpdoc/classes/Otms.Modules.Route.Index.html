    Index class

    Основной для модуля класс, вызывается первый раз при инициализации модуля. Тогда же создаёт необходимые сущности. Затем вызывается из роутера. И в конце postInit() в Bootstrap выполняет окончательные действия: загрузка JS, CSS файлов, передача нужных шаблонов в $this->registry["view"]

    author Dmitry Samotoy
