    CmdCommands Model class

    Класс реализует команды в командной строке. Команда - private функция, например _exit() Аргументы команда получает из массива $this->args[] Команда должна возвращать TRUE в случае успеха, а результат (если есть) содержится в переменной $this->result

    author Dmitry Samotoy
